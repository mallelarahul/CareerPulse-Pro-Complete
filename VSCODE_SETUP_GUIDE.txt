✅ STEP 6: PUSH TO GITHUB

Open Terminal in VS Code (Ctrl + `)

1. Initialize Git:
   git init

2. Add all files:
   git add .

3. Commit:
   git commit -m "Initial commit - CareerPulse Pro"

4. Go to https://github.com/new
5. Create repository: "careerpulse-pro"
6. Copy the URL (looks like: https://github.com/YOUR_USERNAME/careerpulse-pro.git)

7. In Terminal, paste:
   git remote add origin [PASTE_YOUR_URL_HERE]

8. Push to GitHub:
   git branch -M main
   git push -u origin main

9. First push might take 30 seconds
10. Code is now on GitHub! ✅

❓ TROUBLESHOOTING

Issue: "Live Server button not showing"
→ Right-click index.html
→ Scroll down, you'll see "Open with Live Server"

Issue: "Port 5500 already in use"
→ Open settings (Ctrl+,)
→ Search: "Live Server Port"
→ Change to: 5501 or 8000

Issue: "Can't push to GitHub"
Error: "fatal: not a git repository"
→ Make sure you're in correct folder
→ Run: pwd (shows current folder)
→ cd to correct folder if needed

Error: "Authentication failed"
→ Go to https://github.com/settings/tokens
→ Create new token
→ Use that instead of password

Issue: "Jobs not showing"
→ Press F12 (Developer Tools)
→ Click "Console" tab
→ Look for red errors
→ Refresh page (F5)

Issue: "Resume enhancement not working"
→ You need Claude API key
→ But it has fallback - still works!
→ See mock enhancement
